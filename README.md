
# Асноўны гайд і зваротная сувязь:
### Спасылка на артыкул у Steam: [АКТУАЛЬНЫ БЕЛАРУСІЗАТАР ULTRAKILL](https://steamcommunity.com/sharedfiles/filedetails/?id=3671346011&preview=true)
---
### Кароткі гайд на ўстаноўку ПЕРАКЛАДУ ТЭКСТУ і АГУЧКІ:
1. Запампуйце ці купіце ULTRAKILL
2. Запампуйце zip архіў [UltraBel.zip](https://github.com/Ashot-678/ULTRAKILL-Belarusian-Translation/releases/tag/UltraBel) і распакуйце яго ў тэчку ULTRAKILL (выберыце выняць у бягучую тэчку)
3. Каб усталяваць актульны пераклад тэксту, дадаткова запампуйце файл зверху BE-be.json. Шлях усталёўкі: ULTRAKILL\BepInEx\config\ultrakull (Устаўце з заменай)
4. Запускайце гульню
5. Уключыце агучку ў настройках гуку (або выключыце)
6. Калі вы жадаеце выдаліць агучку аднаго пэўнага персанажа, перайдзіце па шляху: ULTRAKILL\BepInEx\config\ultrakull\audio\BEL-bel і выдаліце ​​пэўную тэчку з агучкай персанажа
